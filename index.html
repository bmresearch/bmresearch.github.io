Making the Solana ecosystem a better place for developers, through open source development

What we're building

Solnet

Solnet is Solana's .Net integration library.

This library allows you to integrate Solana into a number of different .Net applications, from desktop applications to video games built on Unity.

Source: GitHub

Status: Live on NuGet

Solnet.Serum

Solnet.Serum is Project Serum's .Net integration library.

Serum is a decentralized exchange and one of the most important primitives for DeFi in Solana.

This library allows you to seamlessly integrate Serum into .Net applications.

Source: GitHub

Status: Live on NuGet

Solnet.Mango

Solnet.Mango is Mango Market's .Net integration library.

Mango Markets provides access to deep liquidity, margin trading and derivatives, built on top of Serum's on-chain CLOB.

This library will allow you to seamlessly integrate Mango into .Net applications.

Status: Under Development.

Solnet.Metaplex

Solnet.Metaplex is Metaplex's .Net integration library.

Metaplex is a protocol for issuance and control of NFTs on Solana, empowering creators and brands to build direct relationships with their audience via self-hosted storefronts.

This library will allow you to seamlessly integrate Metaplex into .Net applications.

Status: Under Development.
